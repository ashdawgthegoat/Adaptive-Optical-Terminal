# Wanderer UI

## Vision

Wanderer UI is the software interface for The Wanderer Project.

It is designed to function as the operating environment for a portable scientific observation terminal.

The user should interact with Wanderer rather than the underlying operating system.

## Design Goals

* Fast startup
* Keyboard-first navigation
* Minimalist interface
* Scientific instrument aesthetic
* Modular architecture
* Offline-first operation

## Planned Boot Sequence

1. Wanderer Project logo
2. Project title screen
3. System initialization logs
4. Main menu

## Main Menu

* Observe
* Archive
* Music
* Modules
* System
