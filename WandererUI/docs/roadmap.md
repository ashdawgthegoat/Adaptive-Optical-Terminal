# WandererUI 1.0 Roadmap

> *A keyboard-first desktop environment built for cyberdecks, terminals and modular computing.*

---

# Guiding Principles

* Keyboard-first interaction.
* Modular architecture.
* Linux-native.
* XDG-compliant.
* Minimal resource usage.
* Native applications share a common panel architecture.
* Existing Linux applications run inside Workbench.
