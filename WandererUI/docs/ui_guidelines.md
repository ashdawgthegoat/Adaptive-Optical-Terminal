# Wanderer UI Design Bible

## Design Oath

Every screen shall feel like operating a scientific instrument,
not using an app.

---

## Five Panel Layout

Header
Navigation
Viewport
Context
Footer

---

## Panel Responsibilities

Header
- Project identity
- Current mode
- Version

Navigation
- Destinations only

Viewport
- Main visual focus
- ASCII
- Camera
- Images
- Graphs

Context
- Screen specific information

Footer
- Controls
- Notifications
- System status

## Layout Rules

Lots of whitespace.

No unnecessary icons.

No decorative widgets.

Every element must answer one question.
